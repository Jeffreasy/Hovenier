# Nu snoeien vs. NIET snoeien

| Nu snoeien (zomerbloeiers) | | NIET snoeien (voorjaarsbloeiers) |
|---|---|---|
| Vlinderstruik (30-50 cm) | | Forsythia |
| Spiraea japonica | | Sering |
| Fuchsia (dood hout) | | Weigela |
| Pluimhortensia (25 cm) | | Boerenhortensia |
| Hagen: half april | | Clematis groep 1 |

> **Broedseizoen:** 15 maart - 15 juli. Check altijd op nesten!

# Snoeien: wat kost het?

| Klus | Kosten |
|------|--------|
| Uurtarief hovenier | **EUR 45-70** |
| Specialist (boomverzorger) | **EUR 60-75** |
| Heg snoeien | EUR 4-8/m |
| Boom klein (tot 3 m) | EUR 150-300 |
| Boom middel (5-7 m) | EUR 300-500 |
| Boom groot (7 m+) | EUR 500-1.500+ |
| Fruitboom | EUR 100-525 |
| Snoeiafval afvoeren | EUR 35-50/m3 |

> **Vuistregel:** Nooit meer dan 1/3 van de plant per seizoen weghalen
